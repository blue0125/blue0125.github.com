用octopress建立博客

发表于 2012-02-27   |   分类于 octopress   |  

分享，其实大多数是只是Octopress文档的翻译而已。
网上也有狠多类是的文章。我也是参考了一些文章后搭建起来的。
就想整理下，自己写出来，锻炼下表达能力。
有错误的地方请指出哈。
还有更多的用法技巧要到以后学习了再来总结 =3。

##安装

###git
先到github注册一个帐号。github的这只猫还是比较可爱的。
注册完帐号后要建立一个用户名.github.com的仓库。
git是一个分布式版本控制系统。是linus(linux核心创作人)发起的另外一个开源项目。

sudo apt-get install git

###ruby
我是用rbenv来安装ruby的。RVM和rbenv的区别说是RVM全面点，rbenv适合我这种外行就搭建个环境用。2个只要安装一个就行。

把rbenv克隆到本地。

$ git clone git://github.com/sstephenson/rbenv.git .rbenv

然后把bin目录加到$PATH。
这样你就可以运行rbenv来安装ruby了。
因为octopress文档上写着需要ruby1.9.2，而且我在一开始用了1.9.3也出点问题(也不知道是不是版本的问题，后来用1.9.2就没错误)。所以这里推荐使用1.9.2。
安装办法有2种。一种是下载源码包后安装、另一种是直接用命令安装。
第一种、安装要放到rbenv的versions目录下。

$ ./configure --prefix=$HOME/.rbenv/versions/1.9.2-p290
$ make
$ make install

第二种、

$rbenv install 1.9.2-p290

安装完后

$ rbenv rehash

###octopress 

$ git clone git clone git://github.com/imathis/octopress.git octopress
$ cd octopress

然后安装依赖的。

$ gem install bundler

这里一些原因(国内的)会导致安装的死慢，可用淘宝的Rubygems镜像。 

$ gem sources --remove http://rubygems.org/
$ gem sources -a http://ruby.taobao.org/
$ gem sources -l
*** CURRENT SOURCES ***

http://ruby.taobao.org
# 请确保只有 ruby.taobao.org
$ gem install foo

重新安装bundler后

$ bundle install

然后安装octopress的默认主题

$ rake install

这里会在octopress根目录下生成source和sass目录。
这样基本就安装好了。

##发布
接下来就编写第一篇博客了。 

rake new_post["post title"]

这里会在source目录下的_post目录下生成一个markdown文件。这个就是要编辑的blog文章了。
接下来就是生成静态站点 

rake generate

这里会读取octopress根目录下的_config.yml的配置文件，并按照source的内容生成public文件的静态站点。 

rake preview

启动本地服务localhost:4000就能看到预览效果了。

先按照github处理好一开始新建的github项目，接着

rake setup_github_pages

按提示输入你的github项目地址，exp:
git@github.com:blue0125/blue0125.github.com.git
这样以后只要输入下面命令就可以发布blog到github上了。

rake deploy

在source下echo 'blue0125.me' > CNAME，用于定制自己的域名，然后把域名的A记录指向207.97.227.245。详见pages.github.com
别忘记再建立一个source项目来备份本地的源文件。
