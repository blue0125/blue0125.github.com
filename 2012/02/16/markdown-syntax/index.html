Markdown-Syntax

发表于 2012-02-16   |   分类于 makrdown   |  

Markdown 的目標是實現「易讀易寫」。
做个小抄，备用。

#Block

##Paragraph Newline
空行或者行末2个空格+Enter

Title

Atx 形式是在行首插入 1 到 6 個 # : 

# This is an H1
## This is an H2 
###### This is an H6

This is an H1

This is an H2

This is an H6

Blockquotes

每行的最前面加上 “>”

List

Markdown 无序和有序列表。 

* 列表1
* 列表2
* 列表3
  • 列表1
  • 列表2
  • 列表3

无序列表使用星号、加号或是减号
有序列表使用数字后面紧跟一个英文句号 

1986\. What a great season.

数字后面的点话，要加反斜杠。

##Code block
缩进 4 个空白或是 1 个 tab: 

This is a normal paragraph:
This is a code block.

##Underline
在一行中用三个或以上的星号、减号、下划线来建立一個分隔线，行內不能有其他东西。 

***

#Inline

##Link
Markdown 支持两种形式的链接语法： 行內和参考两种形式。 

This is [an example](http://example.com/ "Title") inline link.

This is an example inline link.

如果你是要連結到同樣主機的資源，你可以使用相對路徑： 

See my [About](/about/) page for details.

See my About page for details.

##Em Strong
使用星号和下划线来表示强调，1个会转化为<em>，2个会转化为<strong> 

*强调*

强调

__强调__

强调

\*this is * \*

*this is \

##Code
用`把程序代码包含起来

Use the `printf()` function.

Use the printf() function.

##Picture
和链接类似。

![Alt text](/path/to/img.jpg "Optional title")

#Other

##Auto link
支持比较简短的形式来处理网址和邮箱。
link

<http://example.com/>

http://example.com/

mail

<address@example.com>

address@example.com

##Meta character

\ 反斜杠
` 反引号
* 星号
_ 下划线
{} 大括号
[] 方括号
() 括号
# 井字号
+ 加号
- 减号
. 英文句号
! 感叹号
